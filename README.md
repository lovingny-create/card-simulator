# 🎴 보드게임 카드 시뮬레이터

2026 전람회 준비용 반응형 보드게임 카드 전시 프로젝트

## 시뮬레이터

### 1. 카드 갤러리 (`simulator_card_gallery.html`)
- 전체 카드를 그리드로 전시 (액션 60장 + 나무 13장)
- 탭 필터링 (전체/액션/나무)
- 클릭 시 확대 팝업 + 좌우 네비게이션

### 2. 카드 뽑기 (`simulator_card_draw.html`)
- 인터랙티브 카드 드로우 시뮬레이터
- 3D 플립 애니메이션
- 반응형 스케일 (모든 기기 대응)

## 특징
✨ 모든 화면 크기에서 480:686 비율 유지  
📱 태블릿/핸드폰/노트북 최적화  
🚀 Google Sites 임베드 가능  
⚡ 외부 라이브러리 없음 (순수 HTML/CSS/JS)

## 사용 방법
1. HTML 파일을 브라우저로 열기
2. 또는 로컬 서버 실행: `python -m http.server 8000`
3. Google Sites에 임베드: URL을 embed 코드로 추가

## 배포
- GitHub Pages
- Netlify
- 기타 정적 호스팅
